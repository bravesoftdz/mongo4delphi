Mongo Driver Checklist
	http://www.mongodb.org/display/DOCS/Feature+Checklist+for+Mongo+Drivers

Mongo Driver Requirements	- High priority
	* BSON serialization¹
	* BSON deserialization¹
	*	database profiling: set/get profiling level, get profiling info	
	  getProfilingLevel, setProfilingLevel, getProfilingInfo 
	*	advanced connection management (slave okay)
	* Automatically connect to proper server, and failover, when connecting to a Replica Set
	*	automatic reconnection on connection failure
	* Support to GridFS

Medium priority
  * validate a collection in a database
  * Tailable cursor support - http://www.mongodb.org/display/DOCS/Tailable+Cursors
	
Fully implemented:	
	*	full cursor support (e.g. support OP_GET_MORE operation) - OK, needs design review 
	*	close exhausted cursors via OP_KILL_CURSORS - OK
	*	support for running database commands - OK
	*	handle query errors - OK, support to GetLastError
	*	hint - Ok
	* explain - OK
	* count - OK
	* snapshot queries - OK
	* $where - OK
	* authentication - OK
	* Insert - OK
	* Update - OK
	* Remove - OK
	* batch insert - OK
	* batch update - OK, use the operator Multi to modify multiple records
	* batch remove - OK, using a query to remove multiple records
	* Create/Drop Indexes - OK
